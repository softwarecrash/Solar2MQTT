# WebSerial

WebSerial is the quickest way to watch the live device log in the browser.

## What it is used for

- view live log output without a USB serial monitor
- inspect Wi-Fi, MQTT, update and inverter messages in the browser
- copy raw protocol and error lines for issue reports

## How to open it

- Open `Menu`
- Click `Web Serial`

## Important note

If you want to send inverter commands and also see the answer cleanly in the Web UI, use the `Debug` page command field.

## When WebSerial is most useful

- checking protocol detection
- capturing `ERROR Send` lines
- checking update and Wi-Fi reconnection logs
- collecting debug material for unsupported devices
